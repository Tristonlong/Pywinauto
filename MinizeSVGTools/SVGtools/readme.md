使用手册
- 前提条件：Node.js (v14 或更高版本)·
    
- 运行命令：node svg-optimizer.js <目录路径>

- 操作方法： 把需要转换的SVG文件当到Input目录下， 执行运行命令，自动替换压缩好后的SVG文件， 
 注： 多册压缩命令，文件也不会损坏
